# AI Image Prompt Manager

画像生成AIのプロンプトと生成結果を管理・閲覧するためのローカルWebアプリケーション。

## プロジェクト概要

- **目的**: 画像生成AIのプロンプトと結果画像をローカル環境で管理・閲覧する
- **種別**: ローカルWebアプリケーション

## 開発ルール

- コミットメッセージ・PR・コメントは日本語で記述する
- コードのコメントは必要最小限にとどめる
- セキュリティ上の懸念がある情報（APIキー等）はコードに含めない

## Skills

`skills/` ディレクトリに以下のカスタムスキルが配置されている：

- `/code-review` - コードレビューの実施
- `/create-pr` - ghコマンドでPR作成
- `/codex` - OpenAI Codex CLIへの相談・レビュー依頼
